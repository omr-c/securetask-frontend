# SecureTask - Frontend 🚀

SecureTask is the client-side application for the SecureTask ecosystem. Built with **Angular**, it provides a high-performance, reactive dashboard for managing personal tasks with a focus on security and user experience.

## ✨ Key Features

- **JWT Authentication**: Secure login and session management using Interceptors.
- **Task Management**: Full CRUD (Create, Read, Update, Delete) operations.
- **Reactive UI**: Built with **Angular Material** for a professional and consistent look.
- **Dynamic Status Badges**: Real-time visual cues for task progress (Pending, In Progress, Completed).
- **Responsive Design**: Fully optimized for various screen sizes.
- **Material Dialogs**: Modular form handling for better UX.

## 🛠️ Tech Stack

- **Framework**: Angular 18+
- **Styling**: Angular Material, SCSS
- **State Management**: RxJS (Observables)
- **HTTP Client**: Angular HttpClient with Auth Interceptors
- **Icons**: Material Icons
